# FestivalAI

FestivalAI is an AI-powered web application that automatically detects special occasions, festivals, and memorable dates, then generates beautiful greeting images and videos with a premium modern UI.

## ✨ Features

- Detects today's special occasion automatically
- Generates AI-based greeting images
- Supports contextual video previews
- Premium pixel-perfect hero section
- Multilingual-friendly experience
- Clean and modern responsive UI
- Personalized greeting experience for special events

## 🚀 Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## 📌 Project Goal

The goal of FestivalAI is to make celebration content generation fast, beautiful, and personalized. Instead of creating greeting posts manually, users can instantly get high-quality visuals based on the occasion.

## 🖼️ Core Highlights

- Dynamic occasion-based greeting generation
- Premium hero section with modern visual design
- AI image and media integration
- Personalized and scalable UI architecture
